  • GitHub Pages: 무료로 정적 사이트를 호스팅.

필요한 구체적인 기능이나 디자인 요소가 있다면 말씀해 주세요! 원하는 스타일이나 테마를 바탕으로 맞춤 코드를 제공할 수 있습니다. 😊

나의 말:
GitHub Pages 란 무엇인가요
ChatGPT의 말:

GitHub Pages는 GitHub에서 제공하는 무료 정적 웹사이트 호스팅 서비스입니다. 코드를 GitHub 저장소에 업로드하면, GitHub Pages를 사용해 해당 코드를 기반으로 한 웹사이트를 쉽게 배포할 수 있습니다. 주로 개인 프로젝트, 포트폴리오, 문서화된 웹사이트를 배포하는 데 사용됩니다.

GitHub Pages의 주요 특징

  1. 무료: GitHub에서 제공하며, 공개 저장소(Private 저장소도 가능)를 기반으로 무료로 사용할 수 있습니다.
  2. 정적 웹사이트 지원: HTML, CSS, JavaScript 등 정적 콘텐츠를 호스팅합니다.
  3. 자동화된 배포: 저장소에 파일을 푸시하면 자동으로 웹사이트가 업데이트됩니다.
  4. 커스텀 도메인: 원하는 도메인을 연결할 수 있습니다.
  5. Jekyll 지원: Jekyll을 통해 블로그나 정적 사이트를 생성하고 쉽게 배포할 수 있습니다.

GitHub Pages 설정 방법

  1. GitHub 저장소 생성

    • GitHub에 로그인 후, 새로운 저장소를 생성합니다. 예: my-website.

  2. 코드 업로드

    • HTML, CSS, JavaScript 파일을 로컬에서 작성하고 저장소에 업로드합니다.
    • 저장소의 루트 디렉터리 또는 /docs 폴더에 웹 파일을 저장합니다.

  3. GitHub Pages 활성화

    • 저장소로 이동하여 Settings → Pages 메뉴를 클릭합니다.
    • "Branch" 설정에서 배포에 사용할 브랜치를 선택합니다. (일반적으로 main 또는 gh-pages 브랜치)
    • "Save" 버튼을 클릭합니다.

  4. 웹사이트 URL 확인

    • 설정이 완료되면 웹사이트 URL이 표시됩니다.
    • URL은 일반적으로 https://<사용자이름>.github.io/<저장소이름>/ 형태입니다.

  1. GitHub 웹사이트 방문

    • 이메일 주소: 사용할 이메일 주소를 입력합니다.

    • 사용자 이름: 원하는 사용자 이름을 설정합니다. (예: myusername → https://myusername.github.io)

    • "I'm not a robot" 또는 퍼즐을 해결해 캡차를 통과합니다.

  5. 계정 생성

    • "Create account" 버튼을 클릭하여 계정을 만듭니다.

  6. 이메일 인증

    • 입력한 이메일로 인증 링크가 전송됩니다. 이메일을 열어 링크를 클릭하여 인증을 완료합니다.

2. GitHub Pages 설정하기

  1. GitHub 저장소 생성

    • 로그인 후, 오른쪽 상단의 "+" 아이콘을 클릭하고 "New repository"를 선택합니다.
    • 저장소 이름을 입력합니다. (예: my-website)
    • 저장소를 "Public"으로 설정합니다.

  2. 파일 업로드

    • 웹사이트에 필요한 HTML, CSS, JavaScript 파일을 업로드하거나, GitHub의 기본 README.md 파일을 추가합니다.

  3. GitHub Pages 활성화

    • 저장소의 Settings로 이동합니다.
    • 왼쪽 메뉴에서 Pages를 클릭합니다.
    • "Branch"에서 배포 브랜치를 선택합니다. (예: main)
    • 저장을 완료하면 GitHub Pages가 활성화됩니다.

  4. 웹사이트 확인

    • 배포가 완료되면 "GitHub Pages" 섹션에 URL이 표시됩니다. (예: https://<사용자이름>.github.io/<저장소이름>/)

추가 팁

  • 기본 URL 사용: 저장소 이름을 myusername.github.io로 설정하면 서브 디렉터리 없이 https://myusername.github.io로 접속할 수 있습니다.
  • 커스텀 도메인: 사용자 지정 도메인을 연결하려면 DNS 설정에서 GitHub Pages에 연결하세요.
  • 무료 플랜 사용: GitHub은 무료 플랜에서도 GitHub Pages 기능을 제공합니다.
